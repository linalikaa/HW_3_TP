## Как открыть отчёт в браузере в GitHub Codespaces

1. После выполнения `./run.sh report_server` запустится третий контейнер — `nginx:alpine`, 
   который раздаёт файл `data/report.html` как главную страницу (index.html) через порт 80.
2. Откройте вкладку **PORTS**.
3. Найдите порт `8080`
4. Поменяйте видимость порта на **Public** (правый клик по порту → Port Visibility → Public), 
   иначе попросит авторизацию.
5. Нажмите на иконку глобуса/значок "Open in Browser" рядом с портом 8080 — откроется 
   новая вкладка браузера на вашем компьютере с готовым HTML-отчётом.
